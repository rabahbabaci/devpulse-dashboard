# Contributing

Thanks for checking out DevPulse.

## Local development

1. Fork and clone
2. Install dependencies with `npm install`
3. Run with `npm run dev`
4. Open a PR with a clear summary and screenshot/gif for UI changes

## Commit style

Use conventional commit prefixes when possible:

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation-only updates
- `refactor:` non-breaking internal changes

## Quality bar

- Keep TypeScript strict mode green
- Ensure `npm run build` succeeds
- Update README for visible behavior changes
